# Namma-Vastra

## Android App Development using GenAI

---

# 📌 Problem Statement

Local handloom cooperatives, weavers, and clothing donation centers manage large quantities of textiles, but there is no easy way to digitally track their inventory and physical condition. Often, a damaged batch of fabric, color bleeding, or a moth-eaten garment is only discovered right before a sale or distribution. There is no simple way for artisans or managers to audit their textile assets.

**Namma-Vastra** is a **Digital Textile Auditor** designed to simplify textile inventory and quality management. Weavers and cooperative managers can scan or tag clothing batches, monitor quality conditions, and maintain proper documentation for every textile item.

---

# ✨ Features

## 🧵 Textile Register
- Add textile or clothing batches
- Store:
  - Batch Name
  - Weaver/Donor ID
  - Category
  - Photos

### Example:
- Silk Saree Batch A
- Winter Coats
- Cotton Fabric Rolls

---

## 🎨 Quality Update

Routine textile quality monitoring using color indicators:

- 🟢 Green — Premium / Ready
- 🟡 Yellow — Minor Defect
- 🔴 Red — Damaged / Needs Upcycling

---

## 📋 Defect Log

Track damaged or defective items with:
- Date
- Defect Description
- Repair Notes

### Example:
- Thread pull on border
- Color fading
- Stained during transit
- Torn fabric edge

---

## ♻️ Upcycle / Repair Request

Generate a prioritized list of:
- Damaged garments
- Repair requests
- Dyeing requirements
- Upcycling opportunities

---

## 📄 Summary Report

Generate inventory and textile quality reports that can be shared with:
- Buyers
- Cooperative Heads
- Quality Inspectors
- Distribution Teams
